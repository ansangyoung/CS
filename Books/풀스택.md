# 풀스택  
### 저자 김동욱  
#### 1.3 프로그램과 프로세스  
##### 1.3.2 프로그램  
컴파일: 소스코드를 다시 기계어로 번역하는 과정  
컴파일러: 번역을 담당하는 프로그램  
프로그램: 운영체제와 CPU에 종속적인 실행 가능한 파일  

##### 1.3.3 프로세스
heap: 런타임에 동적으로 생성되는 값들  
stack: 함수라고 불리는 프로그램의 반복적인 로직이 실행되고 소멸되는 공간  
프로세스: 운영체제가 메모리에 프로그램을 복사하여 CPU가 처리중인(실행중인) 프로그램  
  
  
#### 1.4 GUI/CLI, Shell, 파일권한  
##### 1.4.1 GUI(Graphical User Interface) 
그래픽을 통해 사용자와 프로그램이 상호작용하는 환경.  
운영체제는 프로그램 개발자가 운영체제 테마에 맞는 공통적인 GUI를 만들 수 있도록 관련 API를 제공하고,  
응용 프로그램은 이를 활용하여 개발되었기에 운영체제 별로 응용프로그램들의 GUI가 비슷하다.  

##### 1.4.2 CLI(Command Line Interface)  
텍스트입출력을 통해 사용자와 프로그램이 상호작용하는 환경  

##### 1.4.3 Shell  
운영체제의 기능을 이용하고 프로그램을 실행하기 위한 프로그램.  
CLI/GUI 두가지 방식으로 제공한다.
